# Front-end do Projeto

## Descrição
Esta é a aplicação front-end do projeto, responsável pela interface com o usuário. Ela consome as APIs do back-end para exibir dados e realizar operações.

## Requisitos
Para executar o front-end, você precisa de:
- Navegador web atualizado (ex.: Chrome, Firefox)

#Execução simples (com navegador):
Abra o arquivo index.html diretamente no navegador.

#Funcionalidades
Interação com API: Consome os dados do back-end usando fetch.
Armazenamento de Dados: Guarda os dados do usuário no banco.
